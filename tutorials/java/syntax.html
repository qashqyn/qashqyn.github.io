Синтакс

В предыдущей главе мы создали файл Java с именем MyClass.java и использовали следующий код для вывода на экран "Hello World":

MyClass.java

public class MyClass{
	public static void main(String[] args){
		System.out.println("Hello World!");
	}
}

Объяснение примера

Каждая строка кода, которая выполняется в Java, должна быть внутри класса. В нашем примере мы назвали класс MyClass. Класс всегда должен начинаться с заглавной первой буквы.

Внимание: Java чувствительна к регистру: "MyClass" и "myclass" имеют различные значение.

Имя java файла должно совпадать с именем класса. При сохранении файла сохраните его, используя имя класса, и добавьте ".java" в конец имени файла. Чтобы запустить приведенный выше пример на своем компьютере, убедитесь, что Java установлена правильно: Перейдите к главе Начало, чтобы узнать, как установить Java. На выводе у вас должно быть:

Hello World
