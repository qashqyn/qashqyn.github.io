Что такое программирование?

Термин программирование означает создать (или разработать) программное обеспечение (software), которое также называется программой. В основных терминах software содержит инструкции, которые говорят компьютеру что делать. Разработчики программного обеспечения создают software с помощью мощных инструментов, называемых языками программирования.

Дальше может быть скучно, но это необходимые знания)

Существует много языков программирования, некоторым из которых уже десятки лет. Каждый язык был изобретен для определенной цели - например, чтобы использовать сильные стороны предыдущего языка или дать программисту новый и уникальный набор инструментов.

Зная, что есть так много доступных языков программирования, было бы естественно задаться вопросом, какой из них лучше. Но, по правде говоря, нет лучшего языка. У каждого свои сильные и слабые стороны.

Опытные программисты знают, что один язык может хорошо работать в одних ситуациях, тогда как другой язык может быть более подходящим в других. По этой причине многие программисты стараются освоить как можно больше разных языков программирования, предоставляя им доступ к обширному арсеналу средств разработки программного обеспечения.
