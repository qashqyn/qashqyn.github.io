Now we have a complete HTML structure with a navbar formed from four parts: logotype, hamburger (menu bar for the mobile platform), navigation links, and social links.
As you see, each navigation link has an icon and text inside. They will fade each other on mouse hover.

All icons are taken from Font Awesom. You should register before using them. Also, you can use another platform as well.

Let's discuss a little bit.

position: fixed; left: 0; top: 0; With this we will fix our navbar on left-top of screen.
position: relative;
position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); With this part of the code, we can center an element horizontally and vertically relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed).

Now, if we open our HTML file in the browser with the attached CSS file, we will see our navbar on top of the screen. The navbar has the only logotype on the left and the menu bar button on the right side.
Our navigation links are hidden, click on the hamburger to show them.
Did you click the button? Is it working? No?

It is all right, button will not work cause we didn't add a script yet.

<script>
    //Hamburger show-hide
    var ham = document.querySelector('.ham');
    var nav = document.querySelector('.nav');
    ham.addEventListener('click', toggleHam);

    function toggleHam() {
        nav.classList.toggle('show');
    }

    //NavLink hover
    document.querySelectorAll('.nav-link').forEach(i => {
        i.addEventListener('click', event => {
            nav.classList.toggle('show');
        })
        i.addEventListener('mouseenter', event => {
            i.querySelector('i').classList.add('fade');
            i.querySelector('i').style.color = 'gold';
            i.querySelector('span').classList.remove('fade');
        })
        i.addEventListener('mouseleave', event => {
            i.querySelector('i').classList.remove('fade');
            i.querySelector('i').style.color = 'white';
            i.querySelector('span').classList.add('fade');
        })
    });
</script>

We added a few Event Listeners like click, mouseenter, and mouseleave. Now button should work.
Also, if you'll position the cursor over one of the navigation link icons, it should fade out and the link's title will appear instead.
